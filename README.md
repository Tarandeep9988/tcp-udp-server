# TCP/UDP Network Server

A Java networking project implementing TCP and UDP socket programming with client-server architecture.

## What's Inside

- TCP Echo Server/Client (port 12345)
- TCP Broadcast Server/Client (port 12346)
- UDP Echo Server/Client (port 9283)

## How to Run

Compile all files:
```bash
javac -d . src/com/network/*.java src/com/network/tcp/echo/*.java src/com/network/tcp/broadcast/*.java src/com/network/udp/*.java
